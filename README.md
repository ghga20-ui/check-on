# 📋 나이스 출결관리 프로 (담임용) 사용 설명서

**매일 아침, 번거로운 출결 서류 작업을 클릭 한 번으로 해결하세요!**
본 프로그램은 담임 선생님들께서 나이스(NEIS)에서 출결 관련 서류를 다운로드하고, 이를 엑셀로 자동 정리하는 과정을 획기적으로 단축해 드립니다.

---

## 🎨 최적의 화면 구성

본 프로그램은 **'나눔바른고딕OTF'** 폰트에 최적화되어 있습니다.
해당 폰트가 설치되어 있지 않더라도 **'맑은 고딕'**으로 깔끔하게 표시되니 안심하고 사용하세요.

---

## 📦 구성 파일 안내

프로그램 배포 폴더에는 아래 파일들이 반드시 **함께** 있어야 합니다.

- 🚀 `NEIS_Teacher_XXXX.exe`: 프로그램 실행 파일
- 📊 `Attendance Result.xlsm`: 출결 결과 자동 정리 엑셀 (매크로 포함)
- 📝 `README.md`: 지금 읽고 계신 사용 설명서

아래 파일들은 **배포 폴더에 포함하면 안 됩니다.**

- `password.txt`
- `password.bin`
- `neis_menu_teacher.json`
- `logs` 폴더

## 🚀 빠른 시작 가이드 (4단계)

### 0단계: 개발 환경에서 직접 실행할 때

처음 실행 전 의존성을 설치합니다.

```bash
pip install -r requirements.txt
```

### 1단계: 우리 학교 공휴일 설정(최초 1회)
1. Attendance Result.xlsm을 연 뒤 '공휴일' 시트에서 공휴일 및 재량휴업일을 입력합니다.
2. 저장 후 종료합니다.


### 2단계: 기본 설정 (최초 1회)

1. **교육청 선택**: 프로그램을 실행 후 본인 소속 교육청을 선택하세요.
2. **인증서 비밀번호**: 나이스 접속 시 사용하는 인증서 비밀번호를 입력합니다. (암호화되어 안전하게 저장됩니다.)
3. **내 반 설정**: 담당하고 계신 학급 번호(1~12반)를 선택하세요.

### 3단계: 기간 및 메뉴 확인

1. **조회 기간**: 서류를 추출할 **연도**와 **월**을 선택합니다.
2. **메뉴 경로**: 학교마다 나이스 메뉴 명칭이 다를 수 있습니다. 프로그램에 설정된 기본값이 실제 나이스 메뉴와 일치하는지 확인하세요.
   - *결석 신고*: 학급담임 → 교육활동신청관리 → 출결관리 → 결석신고서관리
   - *지각·조퇴*: 학급담임 → 교육활동신청관리 → 출결관리 → 지각·조퇴·결과신고서관리
   - *월별 출결*: 학급담임 → 학적 → 출결현황및통계 → 출결현황및통계

### 4단계: 원스톱 자동화 실행 (추천)

- **[🚀 원스톱 자동화]** 버튼을 클릭하세요.
- !!서류를 저장할 폴더를 반드시 새로 만듭니다!!
- 폴더를  선택하면 끝!
- **자동 진행 순서**: 나이스 로그인 → 신고서 다운로드 → 월별 출결현황 다운로드 → 엑셀 매크로 실행 → 최종 결과 생성

## ⚡ 주요 기능 설명

| 기능                           | 설명                                                                  |
| :----------------------------- | :-------------------------------------------------------------------- |
| **🚀 원스톱 자동화**     | 다운로드부터 엑셀 정리까지 전 과정을 한 번에 수행합니다.              |
| **신고서만 다운로드**    | 결석 및 지각/조퇴/결과 신고서(PDF)만 빠르게 내려받습니다.             |

| **엑셀 자동화만 실행**   | 이미 다운로드된 파일들을 기반으로 엑셀 자동 정리 매크로만 실행합니다. |
| **나이스 접속**          | 나이스 메인 화면으로 즉시 이동하여 로그인을 시도합니다.               |

## ⚠️ 주의사항 (꼭 읽어주세요!)

1. **브라우저 조작 금지**: 자동화가 진행되는 동안 크롬 창을 직접 클릭하거나 조작하면 오류가 발생할 수 있습니다.
2. **백업 기능**: 원스톱 자동화 실행 시, 선택한 폴더의 기존 파일들은 자동으로 `backup` 폴더로 이동됩니다.
3. **엑셀 파일 유지**: `Attendance Result.xlsm` 파일의 이름을 변경하거나 삭제하지 마세요.
4. **인터넷 환경**: 학교 보안망이나 인터넷 연결 상태에 따라 나이스 접속 속도가 다를 수 있습니다.
5. **배포 파일 관리**: 사용자 설정 파일(`password.bin`, `neis_menu_teacher.json`)과 로그 파일(`logs/`)은 배포본에 포함하지 마세요.

## 🆘 문제 해결 (FAQ)

**Q. 프로그램이 실행되지 않아요 (Windows Defender 경고)**

- "추가 정보"를 클릭한 후 "실행" 버튼을 누르시면 정상적으로 실행됩니다.

**Q. 로그인이 자꾸 실패해요.**

- 인증서 비밀번호가 나이스에서 사용하는 것과 정확히 일치하는지 확인해 주세요.
- 크롬 브라우저가 최신 버전인지 확인해 주세요.

**Q. 메뉴를 찾을 수 없다는 오류가 떠요.**

- 학교마다 나이스 메뉴의 1~4차 명칭이 조금씩 다를 수 있습니다. 프로그램의 탭 메뉴에서 본인 학교의 실제 메뉴명과 맞게 수정해 주세요.

**Q. 엑셀 결과물이 보이지 않아요.**

- 매크로가 포함된 `Attendance Result.xlsm` 파일이 프로그램과 같은 폴더에 있는지 확인하세요.
- 엑셀 실행 시 "콘텐츠 사용" 또는 "매크로 허용" 메시지가 나오면 반드시 승인해 주셔야 합니다.
